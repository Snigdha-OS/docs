<CardGrid stagger>

      <p>কীভাবে অবদান রাখতে হয় তা জানতে আমাদের <a href="/bn/contributing/">অবদান গাইড</a> দেখুন।</p>
    </Card>
    <Card title="নতুন কন্টেন্ট যোগ করুন" icon="add-document">

      <p>আপনার কন্টেন্ট গঠনের জন্য টিপস পেতে আমাদের <a href="/bn/contributing/">অবদান গাইড</a> দেখুন।</p>
    </Card>
    <Card title="নতুন ভাষা যোগ করুন" icon="setting">

      ```
                    │   └── index.mdx    // হোমপেজের বাংলা সংস্করণ
                    └── ru/
                        └── index.mdx    // প্রয়োজন হলে হোমপেজের রুশ সংস্করণ
      ```
